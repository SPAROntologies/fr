FAIR* Reviews ontology: Description back to ToC

Figure 1 represents the general model describing the FAIR* Reviews ontology. It shows the classes, object properties and data type properties defined in the FAIR* Reviews namespace and how they are aligned with existing ontologies such as FaBiO, BiDO, DataCite from SPAR Ontologies and others like ODRL, FOAF, SKOS or Lexvo.org Ontology.

Some examples of use as well as the pattern followed to use the SKOS concepts and schemes will be soon included.
